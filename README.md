## 🚀 Tính năng chính

* **Tự động hóa dữ liệu (Data Structuring):** Chuyển đổi CV thô (PDF, Web, Text) thành dữ liệu có cấu trúc (JSON) thông qua AI.
* **Hiểu sâu ứng viên (Deep Insight):** Chấm điểm ngữ nghĩa (Semantic Matching), tóm tắt hồ sơ thay vì chỉ tìm từ khóa.
* **Agentic Workflow:** Sử dụng Nanobrowser để tự động xác thực thông tin ứng viên trên LinkedIn/GitHub.
* **End-to-End Automation:** Quy trình khép kín từ Nhận CV -> Phân loại -> Lưu trữ Database -> Hiển thị Dashboard.

## 🛠 Tech Stack

* **AI Engine:** Google Gemini 1.5 Flash.
* **Extension:** Nanobrowser (Chrome Extension Base).
* **Frontend:** React (Vite) .
* **Backend:** Node.js + Express.
